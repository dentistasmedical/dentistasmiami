PROJECT STRUCTURE:
dentistasmiami/
├── app/ (Next.js pages)
├── components/ (React components)
├── content/ (MDX blog posts & neighborhoods)
├── lib/ (utilities & validations)
├── types/ (TypeScript definitions)

KEY FILES:
- types/index.ts - TypeScript interfaces
- lib/utils/index.ts - Helper functions
- lib/validations/contact.ts - Zod schemas
- content/blog/*.mdx - Spanish blog posts
- content/neighborhoods/*.mdx - Neighborhood pages

SECURITY:
- All forms validated with Zod
- XSS prevention via sanitization
- No localStorage (not supported in artifacts)
- Environment variables for API keys
- Server Actions for mutations
