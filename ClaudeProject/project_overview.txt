DentistasMiami.com - Spanish-language dental directory for Miami

GOAL: Generate $1,500 USD/month to fund move to Barranquilla, Colombia

TECH STACK:
- Next.js 15 (App Router)
- TypeScript (strict mode)
- Tailwind CSS v4
- shadcn/ui components
- MDX for content
- Zod for validation
- Google Places API
- Deployed on Vercel (free)

KEY DIFFERENTIATOR: Price transparency (Google doesn't show dental prices)

TARGET USERS:
- Spanish-speaking Miami residents
- Often uninsured/underinsured
- Price-conscious
- Mobile-first
